# Assignment 2 – ARTI 308 (Machine Learning)

## Machine Learning Problem Summary
This project addresses a classification problem in machine learning. The goal is to predict whether a customer will churn (leave the service) based on customer information such as contract type, monthly charges, total charges, tenure, and service options.

### Problem Type
- Supervised Learning
- Classification (because the target is a category: Yes/No)

### Target Variable
- Churn: indicates whether the customer left the company (Yes/No).

### Dataset
The dataset used is the Telco Customer Churn dataset, which includes customer demographic and service features and a churn label.
